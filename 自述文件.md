## 安装

```sh
npm install   ismi-js-tools  --save
```

## pure function

- `typeOf` 校验数据类型
- `lmDebounce` 防抖函数
- `lmThrottle` 节流函数
- `getRandomInt` 获取随机的整数
- `getRandomFloat` 获取随机的浮点数
